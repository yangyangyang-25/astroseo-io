## Why this matters

Static sites are simple to host, but they can still ship missing canonical tags,
thin metadata, or broken internal links.

## Setup

Check page titles, descriptions, canonical URLs, sitemap output, robots rules,
and archive discoverability.

## Implementation notes

Run the audit after every major template change. Template bugs usually affect
many pages at once, so one bad assumption can create site-wide issues.

## Final checklist

- Metadata exists on key pages
- Sitemap and RSS build correctly
- Internal links return 200
