## Why this matters

Broken links hurt readers and can hide important pages from crawlers.

## Setup

Check Markdown links during editing and again after the site is built. Local
paths can behave differently once the base path is applied.

## Implementation notes

Prioritize internal links first because they affect site navigation and content
discovery. Then review external references for accuracy and freshness.

## Final checklist

- Internal links resolve
- External links are still relevant
- Base path behavior is verified
